# 🧮 Modern Calculator

A modern, responsive browser-based calculator built using **HTML5, CSS3, and Vanilla JavaScript**. The application provides a clean and intuitive interface for performing everyday calculations while also offering a built-in formula cheat sheet and calculation history.

## ✨ Features

### 🏠 Landing Page

* Modern home screen with an attractive UI
* Quick navigation to the Calculator
* Formula Cheat Sheet section
* Calculation History section

### 🧮 Calculator

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Decimal number support
* Clear (C) button
* Backspace/Delete (⌫) button
* Division-by-zero error handling
* Operator chaining
* Responsive display for input and results

### 📚 Formula Cheat Sheet

Quick reference formulas organized into categories:

* Arithmetic
* Profit & Loss
* Trigonometric Ratios
* Mensuration (2D & 3D)
* Algebra
* Speed, Time & Distance
* Common Unit Conversions

### 🕒 Calculation History

* Automatically stores successful calculations
* Uses **Local Storage** for persistence
* Displays the latest calculations first
* Delete individual history items
* Clear all history with a single click

### 🎨 User Interface

* Modern glassmorphism-inspired design
* Gradient backgrounds
* Smooth animations and transitions
* CSS Grid-based calculator layout
* Fully responsive across desktop, tablet, and mobile devices

Simply open the `index.html` file in your preferred web browser.
