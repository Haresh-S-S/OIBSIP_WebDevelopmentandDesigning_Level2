# 🏏 MS Dhoni Tribute Page

A modern, responsive tribute website dedicated to **Mahendra Singh Dhoni (MS Dhoni)** — one of the greatest captains and cricketers in the history of Indian cricket.

Designed using only **HTML5, CSS3, and Vanilla JavaScript**, this project showcases a clean, premium UI with smooth animations, responsive layouts, and an engaging presentation of Dhoni's inspiring journey.

---

## 📸 Preview

> *A visually appealing landing page celebrating the life, leadership, achievements, and legacy of MS Dhoni.*

## ✨ Features

- 🎯 Modern full-screen Hero Section
- 🏏 High-quality tribute layout dedicated to MS Dhoni
- 📖 Biography with original, paraphrased content
- 🗓️ Interactive Career Timeline
- 🏆 Achievement Cards
- 💬 Famous Quote Section
- ❤️ Legacy & Tribute Section
- 🎨 Premium UI with gradients, glassmorphism, and animations
- 📱 Fully Responsive Design
- ⬆️ Smooth Scrolling & Back-to-Top Button
- ✨ Scroll Reveal Animations (Vanilla JavaScript)
