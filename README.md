# 🌐 Web Development Tasks Collection

This repository contains four web development projects built as part of a frontend/full-stack learning program. Each project focuses on a different aspect of web development, ranging from responsive UI design to JavaScript interactivity and user authentication.

## 📁 Repository Structure

```
Web-Development-Tasks/
│
├── Task-1-Calculator/
├── Task-2-Tribute-Page/
├── Task-3-ToDo-App/
├── Task-4-Login-Authentication/
└── README.md
```

---

# 📌 Task 1 – Calculator

A modern, browser-based calculator built with Vanilla JavaScript featuring a premium user interface and smooth user experience.

### Features

* Basic arithmetic operations
* Decimal calculations
* Clear and Backspace functionality
* Division-by-zero error handling
* Operator chaining
* Responsive design
* Formula Cheat Sheet
* Calculation History (stored using Local Storage)
* Modern landing page with smooth navigation

**Tech Stack**

* HTML5
* CSS3
* Vanilla JavaScript

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

### Navigate to the repository

```bash
cd <repository-name>
```

Open each task folder separately and follow its setup instructions.

For frontend-only projects (Tasks 1–3), simply open the `index.html` file in your browser or use the VS Code Live Server extension.

For Task 4 (Authentication System):

```bash
cd Task-4-Login-Authentication
npm install
npm start
```

Then open the application in your browser.
